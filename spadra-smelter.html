What It Is

Spadra Smelter is a single-file RF intelligence analysis platform for WiGLE-format wardrive datasets — the standard output format of mobile network scanning tools including the T-Deck Plus running Pisces Moon OS. Drop a CSV, receive a full intelligence picture.

The name is not incidental. Smelting is the process of extracting valuable metal from raw ore. A wardrive CSV is raw ore — thousands of records, most unremarkable, a few significant. Smelter is what turns ore into metal.

What It Does

Interactive Heatmap

Density visualization at city and neighborhood zoom levels. Cluster markers for WiFi and BLE devices at mid-zoom. Powered by Leaflet.js with OpenStreetMap tiles — no API key, no external dependency beyond map tiles.

Anomaly Detection

Mobile hidden networks (high signal strength, no SSID, moving location across sessions). Evil twin candidates (same SSID as common networks with different BSSID). Encryption downgrades (networks previously seen with WPA3 now advertising WPA2). Unusually strong signals at unexpected locations.

OUI Vendor Enrichment

MAC addresses matched against a built-in vendor table. Identifies device manufacturers — distinguishing consumer devices from enterprise equipment, IoT sensors from mobile phones, known vendor profiles from unknown hardware.

Persistent Device Tracking

Tracks devices across multiple wardrive sessions. Identifies devices that appear in the same locations across different dates — infrastructure versus transient devices.

Export

CSV with vendor enrichment data. Printable map view. Plain-text anomaly report.

Architecture

One HTML file. No server. No installation. No backend. No API calls except OpenStreetMap tiles. Runs entirely in the browser. Processes CSV data locally — no data transmitted to any server.

Compatible with any WiGLE-format CSV: WiGLE WiFi Wardriving app, Kismet, NetStumbler exports, and Pisces Moon OS Ghost Engine wardrive output.
