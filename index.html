Embedded Systems & Operating Systems
01
Pisces Moon OS
The first general-purpose OS for the ESP32-S3 hardware class

A dual-core operating system for the LilyGO T-Deck Plus — a $50 microcontroller with WiFi, Bluetooth LE, LoRa mesh radio, GPS, QWERTY keyboard, and touchscreen. Ships 47 applications across 7 categories. Includes a Ghost Partition security system, continuous passive wardriving on Core 0 via the Ghost Engine, local AI inference, and an ELF module runtime for SD card app deployment without reflashing. Six novel engineering problems were encountered and solved during development — none had prior documented solutions for this hardware platform.

Local AI & Intelligence
03
The Phantom
A local AI agent framework — the body the frozen brain never had

A Python wrapper around Ollama that transforms a local language model into a capable intelligence agent. Before each conversation turn, The Phantom scrapes the web, bypasses paywalls, pulls live sports statistics from official APIs, retrieves semantic memories from past conversations via ChromaDB, fetches YouTube transcripts, and reads RSS feeds — injecting all of it as context before the model ever sees the prompt. After the model responds, it saves generated files, distills conversation history, and stages corrections as lessons. The model reasons. The Phantom does everything else. No API keys required for core functionality. No data leaves the machine.

04
WozBot
A live AI service on bare metal — no cloud, no AWS, no term sheet

A publicly accessible AI chatbot running entirely on a MicroPC the size of a paperback book, tunneled to the public internet via Cloudflare. Qwen2.5 1.5B quantized model, llama.cpp inference, Nginx proxy, stateless sessions. No data persists. No user information logged. Demonstrates that a functional public AI service requires neither AWS infrastructure nor venture capital — just a $200 MicroPC, an open-source model, and an afternoon. Named after Steve Wozniak's 1970s Dial-A-Joke phone line.

Publishing Infrastructure
07
Punky
GitHub-native blog editor — no server, no database, no subscription

A single HTML file that serves as a complete blog publishing environment. Write posts using a word-processor-style interface with formatting toolbar, image support, video embeds, and raw HTML injection. On publish: generates a fully-styled HTML post with canonical URL, Open Graph tags, and Schema.org JSON-LD; updates the post manifest; regenerates the sitemap; regenerates the RSS feed — all via the GitHub API, entirely from the browser. Includes a bootstrap utility for migrating existing posts from other editors.

08
Static
GitHub-native podcast publisher — audio on archive.org, pages on GitHub Pages

A single HTML file podcast publishing environment. Add an episode, enter the archive.org audio URL, publish. Static generates the episode HTML page with inline audio player, updates the show homepage with inline players for all episodes, updates the episode manifest, and regenerates a valid RSS feed compatible with Apple Podcasts and Spotify — all via the GitHub API, from the browser. Audio hosted permanently on the Internet Archive. Cannot be deplatformed by a hosting company changing its terms of service.
